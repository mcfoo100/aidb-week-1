Connect Supabase First
This app needs a free Supabase project for cross-device sync and login. Takes 3 minutes.
1. Go to supabase.com → New Project
2. Settings → API → copy Project URL + anon key
3. In Supabase SQL Editor, run:

create table weekend_progress (
  id uuid default gen_random_uuid() primary key,
  user_id uuid references auth.users not null,
  weekend_id text not null,
  done boolean default false,
  quality int default 0,
  time_saved int default 0,
  notes text default '',
  hours_spent numeric default 0,
  updated_at timestamp default now(),
  unique(user_id, weekend_id)
);

alter table weekend_progress enable row level security;

create policy "Users own their data" on weekend_progress
  for all using (auth.uid() = user_id);

4. Paste your credentials below and save
